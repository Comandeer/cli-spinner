Super simple CLI spinner.

## Installation

```bash
npm install @comandeer/cli-spinner --save
```

## Usage

```javascript
import Spinner from '@comandeer/cli-spinner';

const spinner = new Spinner( {
	label: 'Working…'
} );
```
